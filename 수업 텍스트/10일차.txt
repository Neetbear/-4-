함수(function)
    자바스크립트는 함수를 변수에 할당할 수 있으면 인자로 함수를 전달하거나 콜백 함수를 구현할 수 있다.
        콜백함수 = 변수에 함수 할당하는거 등등

    함수 선언 function 키워드() {} 로 선언
        함수 안에서 사용하는 변수는 block-scope인 let이나 const 사용 권장
        함수 안에 새로운 함수 선언 가능

익명 함수 (anonymous)
    변수에 함수를 구현하거나 함수를 return하는 형태
    콜백 함수 구현 가능하게 함

콜백함수(Callback)
    event처리에 많이 사용된다.
    특정 UI에서 이벤트가 발생 되었을 때 처리될 코드를 익명 함수에 넣고 콜백되어 처리될수 있도록 하는 형식
    
동기함수로작용   콜백처리하면 순차적으로 진행이 아니라 동시 진행이 가능하게 해서
                어떤 함수에 timeout을 걸어도 나머진 먼저 뜨고 timeout 걸린거만 일정시간후에 뜬다

    setTimeout함수는 일정시간 후에 동작을 정의 한다.? 일정시간후에 작동된다.

    setInterval 지정한 시간마다 함수 호출


%비동기 함수(asynchronous)
    호출부에서 실행결과를 기다리지 않아도 되는 함수
    어떤 결과를 기다리지 않아도 바로 실행되는 함수
    node.js에서 많이 사용

%반대 개념은 동기 함수(synchronous)
    실행결과가 return 될때까지 기다려야 하는 함수
    설계는 간단하고 직관적이지만 결과가 주어질 때까지 대기

    class(클래스)
        객체 생성때마다 동일한 함수도 매번 생성되므로
         객체선언부와 함수선언부가 분리되어 코드 관리가 불편하다는 점에서 나옴

        클래스 키워드로 선언 / 클래스 이름은 보통 대문자로 시작
        
        class 클래스명 {
            constructor(인자....){
                this.var = 인자;
                ....
            }
            method(){
                return ;
            }
            get매서드{}
            set매서드{}
        }

        클래스 익명 선언 가능 & 변수에 할당도 가능
        new 연산을 통해 호출되는건 클래스 이름이 아니라 생성자 메스드다.
        클래스 바디에는 메서드 선언만 가능 (다른 언어와 같이 멤버 선언은 불가)

        getter, setter
            보통 클래스 필드에 접근할 때는 직접 변수에 접근이 아니라 getter, setter 메서드 사용
            get, set 메서드 정의 가능 / 함수호출이 아닌 클래스의 필드 개념
